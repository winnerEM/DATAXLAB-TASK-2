# Superstore Sales Analysis Dashboard

## Project Overview
This project analyzes Superstore sales data using Power BI to uncover insights on sales performance, customer behavior, product trends, regional performance, and shipping patterns.

## Tools Used
- Power BI
- DAX
- Excel/CSV Dataset

## KPIs
- Total Sales
- Total Orders
- Total Customers
- Average Order Value
- Average Shipping Days

## Key Insights
- The West region generated the highest sales.
- Phones recorded the highest sub-category sales.
- Binders had the highest number of orders.
- Copiers had the highest average order value.
- Standard Class shipping was the most used despite longer delivery time.
